<h2>예술과 삶, 그리고 성찰</h2>
<h4>예술은 인간의 내면을 비추고 연결하는 창입니다</h4>
<h4>당신의 하루를 예술로 표현하고 다른 사람들과 함께 나누면 어떨까요?</h4>
<h4>당신의 감정을 객관적으로 바라보고 예술로 승화시키세요</h4><br>

<h2> 주요 기능 </h2>
- AI 이미지를 생성하여 당신의 하루를 표현하세요<br>  
- 감정 변화 추이를 트래킹하세요<br>
- 당신의 감정을 타인과 함께 나누세요<br>
